list is a base class which has been inherited by MyList
prints an empty list if nothing is passed to it

	>>> MyList = __import__('1-my_list').MyList
        >>> my_list = MyList()
        >>> print(my_list)
        []

prints the list
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> print(my_list)
	[1, 4, 2]

another instance of the sort which actually sort

	>>> my_list = MyList()
	>>> my_list.append(7)
	>>> my_list.append(5)
